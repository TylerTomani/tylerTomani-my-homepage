Part 1 - array list[] and dict{} to data frame

01

Make a DataFrame without numpy for a data frame with a nested list, (each inner list are the values for rows left to right) 

import pandas as pd
data = [[1,4],[2,5],[3,6]]
df = pd.DataFrame(data, index=['row1','row2','row3'],
        columns=['col1','col2'])
df

02

list [] to pd.DataFrame

Make a dict from two list the following way, 

import pandas as pd
states = ['California','Texas','Florida','New York']
population = [39613493, 29730311, 21944577, 19299981]
dict_states = {'States': states,'Population': population}
print(dict_states.items())
print(dict_states.keys())
print(dict_states.values())

03

dict {} to pd.DataFrame

The 'keys' will be the name of the column, the list value for the key will be that columns values 

import pandas as pd
states = ['California','Texas','Florida','New York']
population = [39613493, 29730311, 21944577, 19299981]
dict_states = {'States': states,'Population': population}
df = pd.DataFrame(dict_states)
df
