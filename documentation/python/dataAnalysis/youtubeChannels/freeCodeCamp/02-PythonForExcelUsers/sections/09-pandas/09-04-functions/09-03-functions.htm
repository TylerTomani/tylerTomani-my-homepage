Section 09-04- Functions()

Part 1 - len(), max(), max(df.index) min(), round()

01

len()

The len() function will return the number of rows 

import pandas as pd
df_exams = pd.read_csv('StudentsPerformance.csv')
print(len(df_exams))

02

max()

Will return the the highest numerical or alphabetic value of a a column. 

import pandas as pd
df_exams = pd.read_csv('StudentsPerformance.csv')
print(max(df_exams))

03

max(df.index)

Adding the .index seen in Section 9-2. Attribute attribute to the DataFrame will give us the highest row value 

import pandas as pd
df_exams = pd.read_csv('StudentsPerformance.csv')
print(df_exams.index)
print(max(df_exams.index))

04

min()

You can do the same with minimum as maximum 

import pandas as pd
df_exams = pd.read_csv('StudentsPerformance.csv')
print(df_exams.index)                                
print(min(df_exams))                                
print(min(df_exams.index))

05

round()

Use the round() to round, the 2nd argument is the decimal place to round to,

This csv file only has ints, make a quick DataFrame with floats 

import pandas as pd
float_nums = [3.1, 8.8882, 5.72, 9.9999999]
my_dict = {'floatNumbers': float_nums}
df = pd.DataFrame(my_dict)
print(df)
print(round(df, 2))
