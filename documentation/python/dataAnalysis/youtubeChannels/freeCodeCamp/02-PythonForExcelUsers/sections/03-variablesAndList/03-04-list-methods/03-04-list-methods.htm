Section 03-04- list[] methods()

Part 1 - append() & insert()

01 append()

append() adds its argument to the end of a list 

countries = ['USA','India','China','Brazil']
countries.append('Canada')
print(countries)

02 insert()

insert() will insert the 2nd value at the index from the 1st value. 

countries = ['USA','India','China','Brazil']
countries.insert(0,'Spain')
print(countries)
countries.insert(3,'Cuba')
print(countries)

Notice how the index 'China' and 'USA' will pushed up 1 index

Part 2 - remove() & pop()

01remove()

Enter the value you want to remove 

        countries = ['USA','India','China','Brazil']
        countries.remove('India')
        print(countries)

02 pop('value')

Pop will remove the last element from the list. 

        countries = ['USA','India','China','Brazil']
        countries.pop()
        pop_2nd = countries.pop()
        print(pop_2nd)
        print(countries)

 Notice how it can be also returned into a variable, and 'China' is returned because 'Brazil' was popped() in the previous line.

03 pop(index)

You can also pop() a value from a list with its index, including negative values 

        countries = ['USA','India','China','Brazil']
        america = countries.pop(0)
        print(countries)
        countries = ['USA','India','China','Brazil']
        countries.pop(-2)
        print(countries)
        print(america)
