Section 04-01- Dictionaries

Part 1 - dict keys(), values(), & items()

01

Dictionaries are comprised of key value pairs 

my_dict = { 'name':'Tito', 'age': 34}
print(my_dict)

02 keys()

Get the keys with keys() 

my_dict = {'name':'Tito','age':34}
my_keys = my_dict.keys()
print(my_keys)

03

Get the values with values() 

my_dict = {'name':'Tito','age':34}
my_values = my_dict.values()
print(my_values)

04

Items will return each key value pair into a list[], inside tuples which are Inmutable and cannot be changed 

my_dict = {'name':'Tito','age':34}
my_items = my_dict.items()
print(my_items)
