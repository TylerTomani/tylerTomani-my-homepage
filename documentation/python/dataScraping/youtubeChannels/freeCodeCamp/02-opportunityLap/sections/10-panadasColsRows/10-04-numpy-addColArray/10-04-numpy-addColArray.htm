Part 1 - numpy np.arange() np.random.randint() np.random.uniform()

00

Current csv file

01

To add values to the rows of a column, we must make a numpy array the same length the number of rows.

Use the numpy function arange() with the min and max of the array 

import pandas as pd
import numpy as np
df_exams = pd.read_csv('StudentsPerformance.csv')
language_scores = np.arange(0,len(df_exams))
print(len(language_scores))

02

Set the numpy array as a value to a new or existing column 

import pandas as pd 
import numpy as np
df_exams = pd.read_csv('StudentsPerformance.csv')
language_scores = np.arange(0,len(df_exams))
df_exams['language score'] = language_scores
print(df_exams.head())

03

Create random numbers in the numpy array using np.random.randint()

  • The 1st value is the minimum and is included,
  • The 2nd value is the max but is not included
  • The 3rd is the size
import pandas as pd
import numpy as np
df_exams = pd.read_csv('StudentsPerformance.csv')
language_scores = np.arange(0,len(df_exams))
int_language_score = np.random.randint(0,100, size=len(df_exams))
print(int_language_score.min())
print(int_language_score.max())
print(int_language_score)

Notice the 99 is the maximum value and not 100 proviced into randint()

04

Assign it to the column 

import pandas as pd 
import numpy as np 
df_exams = pd.read_csv('StudentsPerformance.csv')
language_score = np.arange(0,len(df_exams))
int_language_score = np.random.randint(0,100,size=len(language_score))
df_exams['language score'] = int_language_score
print(df_exams.head())

05

Create random floats with random.uniform() 

import pandas as pd
import numpy as np

float_num = np.random.uniform(0,10,size=5)
df = pd.DataFrame()
df['values'] = float_num
