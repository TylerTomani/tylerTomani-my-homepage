01-04 - find() with {'div': class_}

01

The Course title, description, and price are all contained within the <div> with a class of card. We can select the card elements then search query the card for title,description, and price

In find_all add a 1st argument of the element, then a 2nd arguemnet of the selector

make sure you use class_ with a underscore

class is a keyword in python 

from bs4 import BeautifulSoup

with open('home.htm', 'r') as f:
    content = f.read()
    soup = BeautifulSoup(content,'html.parser')
    course_cards = soup.find_all('div', class_='card')
    for card in course_cards:
        print(card.h5.text)
        print(card.a.text)

from bs4 import BeautifulSoup
with open('home.htm','r') as f:
    content = f.read()
    soup = BeautifulSoup(content,'html.parser')
    course_cards = soup.find_all('div',class_='card')
    for card in course_cards:
        course_title = card.h5.text
        course_price = card.a.text.split()[-1]
        print(f'{course_title} cost {course_price}')
