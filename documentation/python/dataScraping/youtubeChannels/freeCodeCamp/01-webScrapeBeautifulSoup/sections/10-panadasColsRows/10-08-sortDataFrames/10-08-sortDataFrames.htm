02

ascending=false

You can omit the 'by' arguement' and just provide the column name. By default, the sort_values will sort rows from lowest to hightest. Assign the ascending paramater to false to change it 

import pandas as pd 
df_exams = pd.read_csv('StudentsPerformance.csv')
df_exams.sort_values('math score', ascending=false)

03

Sort by two different columns with the columns in a list, the first listed takes precedence 

import pandas as pd
df_exams = pd.read_csv('StudentsPerformance.csv')
df_exams.sort_values(['math score','writing score'], ascending=False)

04

Everything we have done so far doens't change the original DataFrame, but instead makes a copy, setting the inplace argument to true, changes the DataFrame.

df_exams.sort_values(['math score', 'writing score'], ascedning=false, inplace=True)
