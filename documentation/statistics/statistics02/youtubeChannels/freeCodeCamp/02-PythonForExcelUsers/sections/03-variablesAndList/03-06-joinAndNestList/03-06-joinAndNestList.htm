Part 2 - Nest list nested_list = [countries, countries_2]

01 append()

Assign the list as values to another list 

countries = ['USA','India','China','Brazil']
countries_2 = ['Italy','Cuba','Germany','Greece']
nested_list = [countries,countries_2]
print(nested_list)

02

You can also append() a list to a variable, The empty nested_list = [] ensures that the countries don't keep being added everytime the cell runs. 

nested_list = []
countries = ['USA','India','China','Brazil']
countries_2 = ['Italy','Cuba','Germany','Greece']
nested_list.append(countries_2)
nested_list.append(countries)
print(nested_list)
