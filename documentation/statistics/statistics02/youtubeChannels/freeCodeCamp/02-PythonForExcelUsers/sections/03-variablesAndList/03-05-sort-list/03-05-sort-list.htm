Part 1 - sort() & sort(reverse=True)

01 sort()

By default sort will go from lowerst to hightest 

countries = ['USA','India','China','Brazil']
num_list = [4, 38, 72, 19]
countries.sort()
num_list.sort()
print(countries)
print(num_list)

01 sort()

Give sort() and argument reverse and set it equal to True 

countries = ['USA','India','China','Brazil']
num_list = [4, 38, 72, 19]
countries.sort()
num_list.sort(reverse=True )
print(countries)
print(num_list)
