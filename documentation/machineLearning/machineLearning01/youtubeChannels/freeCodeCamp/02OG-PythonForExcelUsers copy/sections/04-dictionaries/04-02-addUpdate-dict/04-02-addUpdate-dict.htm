Section 04-02- dict add & update()

Part 1 - add and update() dict

01

Add a key the following way 

my_dict = { 'name':'Tito', 'age': 34}
my_dict['height'] = 5.11
print(my_dict)

02

Update a key with update() method. Give update() the key value pair with its new value 

my_dict = { 'name':'Tito', 'age': 34}
my_dict['height'] = 5.11
print(my_dict)
my_dict.update({'height':2})
print(my_dict)
