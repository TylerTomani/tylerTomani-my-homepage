Section 01-04- Common Shortcuts

Part 1 - Common Shortcuts

01

Find and Replace

Press "F" while in command mode (cell will be blue)

This allows you to find and replace values within the cell

02

⌘ + Enter

Will run the cell,

⇧ + Enter

Runs the cell but creates a new cell below too

03

Number the lines by pressing,

"L"

in command mode(blue) to toggle line number

Part 2 - Heading Sizes

01

Select and markdown cell with a title in command mode (blue), then hit the numbers 1- 9 to increase or decrease size
