Section 03-03- list[start:stop] slicing

Part 1 - list [start:stop] slicing

01

 Put the index you want to start from 1st, then where you want to stop 2nd. 

countries = ['USA','India','China','Brazil']
sliced = countries[1:3]
print(sliced)

 You can return from the slice [start:stop] in to another variable

02

Enter only a value for [:stop] to get the value up to that index 

countries = ['USA','Inida','China','Brazil']
print(countries[:3])

03

Put a [start:] value only, at get from there to the end 

countries = ['USA','Inida','China','Brazil']
print(countries[2:])
