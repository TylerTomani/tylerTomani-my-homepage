Section 03-07- update & copylist[]

Part 1 - update list & copy with slice list[:]

01 list[0] =

Use the index to change the value in a list 

num_list = [5, 79, 33, 58]
num_list[0] = 200
print(num_list)

Part 2 - copy with slice list[:] & .copy()

02 copy with slice list[:]

Use slice with no arguments to copy a list 

countries = ['USA','India','China','Brazil']
copy_list = countries[:]
print(copy_list)

02

Or use the copy() method 

countries = ['USA','India','China','Brazil']
copy_list = countries.copy()
print(copy_list)
