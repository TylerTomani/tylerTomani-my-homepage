Section 04-04- remove dict elements

Part 1 - pop(), del, clear()

01-pop()

We can use pop(), and insert the key we want to remove as it's argument. 

my_dict = { 'name':'Tito', 'age': 34, 'height': 5.11}
popped_key = my_dict.pop('name')
print(popped_key)
print(my_dict)

Remember, pop can also return the popped element

02 del

We can use the del keyword followed by the dict with the key we wish to delete. 

my_dict = { 'name':'Tito', 'age': 34, 'height': 5.11}
del my_dict['age']
print(my_dict)

03-clear()

Use clear() to empty a dict 

my_dict = { 'name':'Tito', 'age': 34, 'height': 5.11}
my_dict.clear()
print(my_dict)
