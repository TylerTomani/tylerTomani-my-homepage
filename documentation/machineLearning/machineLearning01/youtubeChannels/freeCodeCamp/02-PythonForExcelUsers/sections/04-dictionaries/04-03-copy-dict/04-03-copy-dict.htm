Section 04-03- copy dict

Part 1 - copy()

01

Just like list, we can use the copy() method 

my_dict = { 'name':'Tito', 'age': 34}
dict_copy = my_dict.copy()
print(dict_copy)

If we change the original data in my_dict, the data in dict_copy will NOT change

02

If you just assign the old dict to a new dict, then change the data in the original, the data in the new assigned dict will also change 

old_dict = {'name':'Henry','age': 28 }
new_dict = old_dict
print(new_dict)
old_dict.update({'age': 1 })
print(new_dict)
