06-01b- *args and **kwargs

*args

We can pass multiple arguements using the *args keyword as a parameter. The value will be returned as a tuple 

def performOperation(*args):
    return(args)
performOperation(1,2,3)

This can easily be casted into a list 

def perfomOperation(*args):
    return(list(args))
performOperation(1,2,3)

**kwargs

*args passes multiple positional arguements and returns them as a tuple, **kwargs stands for keyword arguments and will return them as a dictionary. 

def perfomeOperation(*args,**kwargs):
    print(args)
    print(kwargs)

perfomeOperation(1,2,3,operation = 'message')
