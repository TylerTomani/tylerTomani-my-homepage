10-2-2- csv.DictReader

 As seen in the previous lesson, we can skip the header using the next() function within with open(...), Or by casting into a list with list(csv.reader(f, delimiter='\t').

 But the csv.DictReader allows us to turn the items of the header row into values for keys within a dictionary. Look at the original file then the code below and see how the header items are now keys for each column. 

import csv

with open('../10_02_us.csv','r') as f:
    reader = csv.DictReader(f, delimiter='\t')
    for row in reader:
        print(row)
