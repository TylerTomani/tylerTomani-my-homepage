10-2-3- filter Data

In the previous lesson we used the csv.DictReader to covert the header row items into keys for dictionaries. We can cast these dictionaries into a list and filter out specific values from those keys.

The below example filters out any postal code that is a prime number and also in the state of 'MA'. 

import csv
with open('../10_02_us.csv','r') as f:
    data = list(csv.DictReader(f, delimiter='\t'))

primes = []
for n in range(2,99999):
    for factor in range(2,int(n**0.5)):
        if n % factor == 0:
            break
    else:
        primes.append(n)

data = [row for row in data if int(row['postal code']) in primes and row['state code'] == 'MA']
print(len(data))

We can see that it works by checking the len. There are 91 results. Go to the next lesson to see how to write the data to a file.
