10-2-4- Writing

We need to use the code from the previous lesson to write the data into a file. 

import csv
with open('../10_02_us.csv','r') as f:
    data = list(csv.DictReader(f, delimiter='\t'))
primes = []
for n in range(2,99999):
    for factor in range(2,int(n**0.5)):
        if n % factor == 0:
            break
    else:
        primes.append(n)
data = [row for row in data if int(row['postal code']) in primes and row['state code'] == 'MA']
print(len(data))

The data gathered from the code above contains all the MA real estate with postal codes that are prime numbers. We can write whatever we want from those dictionaries into a new file, in this case we are writing the 'place name' and 'county'

To do this we must use the writer() function from csv and writerow() function 

with open('primeEstate.csv','w') as f:
    writer = csv.writer(f)
    for row in data:
        writer.writerow([row['place name'], row['county']])

We could pass a delimiter argument to specify how we want the content spaced, but default it is a comma which is what we want, this is what the final output should look like, and should be in the same folder as the script.
