10-3-1 json.loads()

This is a json string, notice it is a string and Not a python dictionary.

jsonString = '{ "a" : "apple", "b": "bear", "c": "cat" }'

To convert it into a python dictionary we must import the json module to use the json method loads() 

import json
jsonString = '{ "a" : "apple", "b": "bear", "c": "cat" }'
json.loads(jsonString)

In VS code or any other IDE, just assign this to a variable to see the results

Adding a trailing comma to a python dictionary is fine, 

import json
jsonString = '{"a":"apple","b":"bear","c":"cat"}'
jsonDict = json.loads(jsonString)
jsonDict = {'a':'apple','b':'bear','c':'cat',}
print(jsonDict)

But adding a trailing comma to a json string will return a JSONDecodeError. 

import json
jsonString = '{"a":"apple","b":"bear","c":"cat",}'
jsonDict = json.loads(jsonString)
jsonDict = {'a':'apple','b':'bear','c':'cat'}
print(jsonDict)

It is common to use a try and except to catch this JSONDecodeError when using json. But you also need to import from the json module itself 

import json
from json import JSONDecodeError
jsonString = '{"a":"apple","b":"bear","c":"cat",}'
try:
    json.loads(jsonString)
except JSONDecodeError:
    print('Bad json string')
