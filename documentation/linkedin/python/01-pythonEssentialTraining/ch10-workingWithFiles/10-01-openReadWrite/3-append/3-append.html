3-append

  As seen in the previous lesson, when we use f = open ('newFile.txt','w') then f.write('Line 1\n')...
 It will overwrite any existing content in the file.

 To fix this, open an existing file you wish to add content to in append mode by using a as the second argument in open().
Then add content with f.write() 

f = open('newFile.txt','a')
f.write('Line 3 \n')
f.write('Line 4 \n')
f.close()
