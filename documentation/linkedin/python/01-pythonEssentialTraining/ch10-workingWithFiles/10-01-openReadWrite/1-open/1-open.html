10-01-1- Opening Files

 To open a file assign the open() function with the file you wish to open as the first argument and 'r' as the second for read. The file you want to open must be in the same directory as the python script. Or the path should be supplied to its location

Printing f will display a file object 

f = open('10_01_file.txt','r')

We can use f.readline() to read one line at a time 

f = open('10_01_file.txt','r')
f.readline()

If we run it again it will display the following line 

f = open('10_01_file.txt','r')
f.readline()
f.readline()

f.readlines()

f.readlines() will place each line of the file as an item into a list including the \n line character. 

f = open('10_01_file.txt','r')
f.readlines()

one way to display each line would by looping through the f.readline() function 

f = open('10_01_file.txt','r')
for line in f.readlines():
    print(line)

strip()

We can use the strip() function to get rid of the new lines or any trailing white space 

f = open('10_01_file.txt','r'):
for line in f.readlines():
    print(line.strip())
