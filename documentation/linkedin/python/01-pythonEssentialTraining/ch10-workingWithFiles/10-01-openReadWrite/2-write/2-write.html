2-write

 To write to a file we assign a variable to the open() function. The first argument is the file name we want to create, and the second argument is w for write

 When we run the file below it will create a file with the name supplied in the same folder as the python script

Important Note In Visual Studio Code you must run the script in 'Run Current File in Interactive Window'. I made the shortcut,

⌃ + ⌘ + \ 
f = open('newFile.txt','w')
print(f)

Then the file will appear in the same folder as the python script

 Use the write() function to write to the file. Important note Always close the file after writing with close()

 When we open the file we will the see the two lines appear directly next to each other

  Important Note, You can just run the script normally in VS code to write to the file, the shortcut for my settings is

⌥ + ⌘ + \ 
f = open('newFile.txt','w')
f.write('Line 1')
f.write('Line 2')
f.close()

 You must add the \n new line character for the lines to be written as such

And Every time you write to a file, it overwites any existing content already in the file so Be Careful

Important if it doesn't work with the run script shortcut. Use Run Current File in Interactive Window 

f = open('newFile.txt','w')
f.write('Line 1 \n')
f.write('Line 2 \n')
f.close()
