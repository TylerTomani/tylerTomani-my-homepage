06-02-Variables and Scope

locals()

Python has a built-in functions locals() that will return the local variables within a scope into a dictionary. 

def performOperation(num1,num2,operator = 'sum'):
    print(locals())

performOperation(1,2,operator = 'multiply')

globals()

The globals() function will print out the global variables, by default python has many global variables

globals()

mixing global and local

If we define a gobal varaible varA globally, then use another varA as a parameter to a different scope, the two will be different 

message = 'A message'
varA = 8
def function1(varA,varB):
    print(message)
    print(varA)
    print(locals())

def function2(varC,varD):
    print(message)
    print(varA)
    print(locals())

function1(1,2)
function2(5,6)
