default value for parameters

Assign a default value in the parameters 

def performOperation(num1,num2,operator = 'sum'):
    if operator == 'sum':
        return num1 + num2
    if operator == 'multiply':
        return num1 * num2
result = performOperation(2,5)
print(result)

Override the the default parameter by assigning the keyword with a different value. You do not need to use the parameter name but you should when there are many. 

def performOperation(num1,num2,operator = 'sum'):
    if operator == 'sum':
        return num1 + num2
    if operator == 'multiply':
        return num1 * num2
result = performOperation(2,5,operator = 'multiply')
print(result)

The keyword arguments order doesn't matter, this is why it is good practice to assign the value with the name name of the keyword 

def performOperation(num1,num2,operator = 'sum', message='default message'):
    print(message)
    if operator == 'sum':
        return num1 + num2
    if operator == 'multiply':
        return num1 * num2
result = performOperation(2,5,message = 'new message',operator = 'multiply')
print(result)

The positional arguments have to be first, in the above example num1 & num2, the keyword arguments have to come after the positional argument

*args & **kwargs

*args allows us to pass multiple positional arguements and will return them as a tuple, **kwargs stands for keyword arguments and will return them as a dictionary. 
def perfomeOperation(*args,**kwargs):
    print(args)
    print(kwargs)

perfomeOperation(1,2,3,operation = 'message')
Python has a built in sum argument which will add and return the arguents passed in. *args We have to import the math library to get a product using the **kwargs 
import math
def perfomOperation(*args,operator = 'sum'):
    if operator == 'sum':
        return sum(args)
    if operator == 'multiply':
        return math.prod(args)
result1 = perfomOperation(1,5,8)
result2 = perfomOperation(1,5,8,operator='multiply')
print(result1)
print(result2)
