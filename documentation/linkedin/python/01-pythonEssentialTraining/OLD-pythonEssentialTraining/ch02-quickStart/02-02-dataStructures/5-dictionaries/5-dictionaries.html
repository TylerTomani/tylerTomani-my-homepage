02-02-5_Dictionaries

 Dictionaries are very different from the previous structures but are used often, If you are familiar with Javascript they are very similar to Objects.

 Declare a dictionary by using {} curley braces but inside of the parenthesis we insert "key" "value" pairs 

my_dict = {
    'key': 'value1',
    'key2': 'value2'
}

We can retrieve values from keys by typing the dictionarie's name and enclosing a "key" name within [] square brackets 

my_dict = {
    'apple': 'a red fruit',
    'bear': 'scary animal'
}
print(my_dict['apple'])

The keys in dictionaries must have unique names, otherwise when attempting to retrieve a value of a key, it will return the last occurance of the name of the key that occurs twice. 

my_dict = {
    'apple': 'a red fruit',
    'bear': 'scary animal',
    'apple': 'delicious'
}
print(my_dict['apple'])
