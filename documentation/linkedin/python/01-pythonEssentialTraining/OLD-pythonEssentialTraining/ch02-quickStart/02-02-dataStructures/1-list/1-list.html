02-02-1_Lists

To create an empty list use [ ] square brackets

my_list = [ ]

Add items to a list by seperating each one with a , comma. 

my_list = [1,2,3,4]
print(my_list)

Different types of variables can go into a list

my_list = ['list', 'of', 'things']
print(my_list)

You can mix different types of varialbe within a list, even other list. 

my_list = [1, 'list', False, ['one','two']]
print(my_list)

It is also possible to put list in list 

my_list = [[1,2], [False, True], ['three','four']]
print(my_list)
