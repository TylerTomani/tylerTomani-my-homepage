02-02-4_Tuples

Tuples use ( ) parenthesis and are simiilar to list; they have a length len() but, they can not be altered or changed. 

my_tuple = (1,2,3)
print(len(my_tuple))

Order also matters with tuples

print((1,2) == (2,1))

It will return False

append()

A major difference between tuples and list is that you can not append or add things to tuples

Use the append() method to a list 

my_list = [[1,2,3],[False,True],4,5]
print(my_list)
my_list.append(6)
print(my_list)

Notice how append() does not work on tuples. You will receive a Attribute Error 

my_tuple = (1,2,3)
my_tuple.append(4)
print(my_tuple)
