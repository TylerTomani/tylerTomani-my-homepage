5-membershipOperators

There are only two Membership operators

  • in
  • not in

in

Checks for a value in a structure or variable.

print(1 in [1,2,3])

If it's not there it returns False

print(8 in [1,2,3])

not in

Use not it to return True if the value is not there.

print(10 not in [1,2,3])

This can be applied to strings. You can search for letters and words in strings.

print('cat' in 'Concatenation')

Or you can retrieve a True by using not in
