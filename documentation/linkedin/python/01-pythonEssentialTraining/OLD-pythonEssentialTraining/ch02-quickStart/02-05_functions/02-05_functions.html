 Functions are defined with the def key word before whatever name we give to the function.

 after the name of the function we place () parenthesis where we can enter variables and values to be used in the function.

 Like if statements and loops, the block of code must be indented on the following line after the :

We can return values from functions with a return statement. 

def myFunction(value):
    return

Invoke the function by typing the name somewhere in the program with () parenthesis after the name 

def sayHey(string):
    return string * 3

sayHey('hey')

Notice how this doesn't do anything if we run the script. We want the word 'hey' three times, but the sayHey() function is only returning the value, with no where to go.

There are two ways to retrieve the value from the sayHey function.

Invoke the function inside of a variable , then print the variable. 

def sayHey(string):
    return string * 3
heys = sayHey('hey')
print(heys)

Or

Or you can Invoke the sayHey() function as a value for the print() function 

def sayHey(string):
    return string * 3

print(sayHey('hey '))
