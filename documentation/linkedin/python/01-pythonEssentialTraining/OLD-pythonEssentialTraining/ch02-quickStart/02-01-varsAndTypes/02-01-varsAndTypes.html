Strings

You can use '' single quotes or double quotes for strings. Nowadays it's standarad practice to use '' single quotes. You can use the + operator to concatenate strings together. 

print('String1' + 'String1')
print('1 ' + '1')

If you add two numbers written as strings, it will concatenate them and not add them.

If you attempt to add a number written as a string with a number, you will receive a TypeError.

print('1' + 1)
