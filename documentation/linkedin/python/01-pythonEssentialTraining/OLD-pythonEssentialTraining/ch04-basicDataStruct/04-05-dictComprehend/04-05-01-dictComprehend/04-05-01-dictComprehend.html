04-05-01-dictComprehend

The following converts a list of tuples into a dictionary with key value pairs 

animalsList = [('a', 'aardvark'),('b','bear'),('c','cat'),('d','dog')]
animalsDict = {item[0]: item[1] for item in animalsList}
animalsDict
