This is list comprehension, taking each element of a list or string and putting it into a new list.

[myFunction(el) for el in myList]

We can pass each element from a list or string into a function the same way. 

myString = 'My name is Teddy Truman. I live in Arizona'
def toLower(word):
    return word.lower()
[toLower(item) for item in myString]

 The removeDots function uses the replace() function to remove the first argument '.' and replace it with its second '', which is a blank string/nothing.

It then lowercases each letter with lower(). This is called chaining methods/functions 

myString = 'This is a String. It Has periods.'
def removeDots(wrd):
    return wrd.replace('.','').lower()
[removeDots(word) for word in myString.split()]
