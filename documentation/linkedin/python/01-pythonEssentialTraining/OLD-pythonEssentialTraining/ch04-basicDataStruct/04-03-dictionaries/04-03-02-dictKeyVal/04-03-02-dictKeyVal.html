04-03-02 dict_keys() & dict_values()

The dict_keys and dict_values object are immutable, but appendable.

To change a dict key or value change the underlying dictionary.

You can convert dict_keys and dict_values into list [] 

animals = {
    'a': 'aardvark',
    'b': 'bear',
    'c': 'cat',
    'd': 'dog'
}
list(animals.keys())
list(animals.values())
