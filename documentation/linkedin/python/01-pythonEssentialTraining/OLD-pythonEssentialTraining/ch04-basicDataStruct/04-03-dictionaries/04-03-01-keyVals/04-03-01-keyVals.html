04-03-01 keys() & values()

Use .keys() on an dictionary to list the keys. 

animals = {
    'a': 'antelope',
    'b': 'bear',
    'c': 'cat',
    'dog': 'dog'
}
animals.keys()

Use .values() 

animals = {
    'a': 'antelope',
    'b': 'bear',
    'c': 'cat',
    'dog': 'dog'
}
animals.values()

Notice how .keys and .values() returns a dict_keys
