04-03b-defaultdict

Import the defaultdict from the collections module/library

from collections import defaultdict

You must past the name of the type of object the default dict must return by default, in this example passes list into it. 

animals = defaultdict(list)
animals

This has created an empty dictionary. Since list is the default value within the dict, we don't have to create keys with empty list ( as seen at the end of the last lesson)

We can just use append values to list in keys, even if the key doesn't exist, it will create the key with the appended list 

animals['b'].append('boar')
animals

Now if we index a key that doesn't exist, no error occurs; Instead a new key is made with an empty list 

animals['f']
animals
