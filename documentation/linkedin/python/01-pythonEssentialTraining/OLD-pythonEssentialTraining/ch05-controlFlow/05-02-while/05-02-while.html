05-02 while loops

Import the class datetime for these examples

Make a while loop that prints a message while waiting, then when the while loop is done print the current seconds 

from datetime import date time
wait_until = datetime.now().second + 2

while datetime.now().second != wait_until:
print("waiting")
print(f'We are at {wait_until} seconds')

pass

The above will not work it will print waiting indefintly

To fix use a pass statement 

from datetime import datetime
wait_unitl = datetime.now().second + 2
while datetime.now().second != wait_unitl:
    pass
print(f'We are at {wait_unitl} seconds')

break

from datetime import datetime
wait_until = datetime.now().second + 2
while True:
    if datetime.now().second == wait_unitl:
        print(f'We are at {wait_unitl} seconds')
        break

The break only breaks out of the first while loop it encounters, in this example it is the while True statement.

continue

continue will skip any lines of code that follows it inside of the while loop. 

from datetime import date time
wait_unitl = datetime.now().second + 2
while datetime.now().second != wait_unitl:
    continue
    print("waiting")
print(f'We are at {wait_unitl} seconds')

This isn't the best way to use continue statement. It's best used for if statement when condition are not met 

from datetime import datetime
wait_until = datetime.now().second + 2
while True:
    if datetime.now().second < wait_unitl:
        continue
    break
print(f'We are at {wait_unitl} seconds')
