continue

Use the continue method to skip a block of code. This example skips any list containing more than one animal and prints only the first animal in the list. 

animalLookup = {
'a': ['aardvark','antelop'],
'b': ['bear'],
'c': ['cat'],
'd': ['dog']
}
for letter,animals in animalLookup.items():
    if len(animals) > 1:
        continue
    print(f'first animal in list: {animals}')

break

We can break out of a loop if we find what we are looking for. So in this example, the loop will print any list with more that one item 

animalLookup = {
'a': ['aardvark','antelop'],
'b': ['bear'],
'c': ['cat'],
'd': ['dog']
}
for letter,animals in animalLookup.items():
    if len(animals) > 1:
        print(f'length: {len(animals)} animals: {animals}')
        break

for/else prime numbers

for number in range(2,50):
    for factor in range(2, int(number**0.5)):
        if number % factor == 0:
            break
    else:
        print(f'{number} is prime')
