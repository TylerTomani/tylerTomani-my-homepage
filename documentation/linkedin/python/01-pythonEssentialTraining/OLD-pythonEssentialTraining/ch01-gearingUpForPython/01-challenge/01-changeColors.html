This page explains how to install and use termcolor within python scripts/

To instal it enter one of the two commands below in the terminal.

python3 -m pip install --upgrade termcolor 
    git clone https://github.com/termcolor/termcolor
    cd termcolor
    python3 -m pip install .

To see an example after installing right away, run the command

python3 -m termcolor

 In the Example section of the page, it shows how to use additional attributes such as reverse and blinkattrs=["reverse","blink]
