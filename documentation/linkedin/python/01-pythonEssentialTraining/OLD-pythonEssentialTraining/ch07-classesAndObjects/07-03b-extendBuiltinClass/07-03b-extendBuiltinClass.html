07-03b- Extending built-in classes

 Remember, python classes, such as list are lowercase but are still classes. We can extend these built-in classes and add atttributes and methods.

 If we want to make a list that makes all the items unique like a set, we would have to extend the list class and overwrite the append() function. We could try the following, 

class UniqueList(list):
    def append(self,item):
        if item in self:
            return
        self.append(item)

 This is will not work, This append method will call itself's self and lead to infinite recursion.

We can call the parent's append method with super() 

class UniqueList(list):
    def append(self,item):
        if item in self:
            return 
        super().append(item)
                
class UniqueList(list):
    def append(self,item):
        if item in self:
            return 
        self().append(item)
uniqueList = UniqueList()
uniqueList.append(1)
uniqueList.append(1)
uniqueList.append(2)

The parent's append class method works as will the child's.

Initializing __init__ in childe

class UniqueList(list):
    def __init__(self):
        self.someProperty = "another list"
    def append(item)
        if item in self:
            return 
        super().append()

Using __init__ in a child class will overwrite the parent's class __init__ initializing method.

To solve this use super().__init__() in the child's __init__ method. 

class UniqueList(list);
    def __init__(self):
        super().__init__()
        self.someProperty = 'another list'
    def append(self,item):
        if item in self:
            return
        super().append(item)

uniqueList = UniqueList()
uniqueList.append(1)
uniqueList.append(1)
uniqueList.append(1)
print(uniqueList)
print(uniqueList.someProperty)
