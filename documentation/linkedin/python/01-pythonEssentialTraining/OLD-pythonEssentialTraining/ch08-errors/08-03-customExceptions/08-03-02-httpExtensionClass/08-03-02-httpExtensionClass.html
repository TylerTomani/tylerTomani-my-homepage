08-03-02- http Exception class

This create a generic HttpException extended from the Exception class. Review chapter 07-03b-Extending built-in classes. 

class HttpException(Exception):
    statusCode = None
    message = None
    def __init__(self):
        super().__init__(f'{self.statusCode} : {self.message}')

We can then create child classes from the HttpException to target specific common http errors by extending HttpException and initializing the variables. 

class NotFound(HttpException):
    statusCode = 404
    message = 'Resource Not found'

class ServerError(HttpException):
    statusCode = 500
    message = 'The server messed up'

To check if it works create a function that raises one of the child classes. 

def raiseServerError():
    raise ServerError
