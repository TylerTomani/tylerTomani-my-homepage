08-02-04 Custom Decorators

It's best practice to create a function to handle errors and use it as a decorator. You must place the excepts into a inner function, this one is called wrapper. we used @staticmethod decorator in chapter 7-02 static and instance (go back and fix that lesson) 

def handleExceptions(func)
    def wrapper():
        try:
            func()
        except TypeError:
            print('type error')
        except ZeroDivisionError:
            print('zero division error')
        except Exception:
            print('error')
    return wrapper
@handleExceptions

 To use the handleExceptions() function as a decorator type @ followed by the name of the function @handleExceptions

The function causeError below the decorator will be passed into the @handleExceptions function as the argument (func ) 

def handleExceptions(func):
    def wrapper():
        try:
            func()
        except TypeError:
            print('type error')
        except ZeroDivisionError:
            print('zero division error')
        except Exception:
            print('error')
    return wrapper


@handleExceptions
def causeError():
