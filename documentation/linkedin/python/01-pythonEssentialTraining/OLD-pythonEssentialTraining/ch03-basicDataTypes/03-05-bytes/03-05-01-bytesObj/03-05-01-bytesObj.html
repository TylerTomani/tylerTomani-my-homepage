bytes data are immutable

Just like tuples, bytes objects are immutable. To mutate and change bytes varaibles, use bytearray. 

sunglasses = bytearray('ðŸ˜Ž','utf-8')
print(sunglasses)

Now we can see that the varaible is a bytearray and not a byte object

Now that the sunglasases varaible is a bytesarray object we can mutate it.

 We can index a bytearray like a string or list. Each hexadecimal is an index. sunglasses[3] is b'\xf0\x9f\x98\x82

Assign sunglasses[3] a value by casting a number with int() class with base 16.

sunglasses[3] = int('82',16)

Decode it to see the change. 

sunglasses[3] = int('82',16)
sunglasses.decode('utf-8')
