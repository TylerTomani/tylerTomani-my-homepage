03-03-3 Boolean Logic

We can evaluate a boolean without the bool() class by typing it alone. 

my_list = [1,2]
if my_list:
    print('list has values')

This is exactly the same as using bool(my_list) 

my_list = [1,2]
if bool(my_list):
    print('list has values')

We can also evaluate expressions as booleans 

a = 2
b = 2
if a - b:
    print('not zero')
else:
    print('zero, false')
