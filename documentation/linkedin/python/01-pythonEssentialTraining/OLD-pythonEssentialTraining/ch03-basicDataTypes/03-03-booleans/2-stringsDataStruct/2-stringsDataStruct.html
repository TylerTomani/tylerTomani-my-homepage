03-03-2 strings and data structs

Strings

The string 'True' will return true in bool()

bool('True')

As will the string 'False'

bool('False')

Only an empty string will return false

bool('')

Even if there is a space, it will return true. Only empty strings return false

bool(' ')

List

The same principles apply to data structures;

  • list
  • sets
  • tuples
  • dictionaries

bool() will return them as True if they contain values, and False if they contain nothing

bool([])
bool([1,2])

The None value you functions

 Remember, the None keyword is returned from any function with no return value. Review Chapter 2 5.Functions

This none value returns False

bool(None)
