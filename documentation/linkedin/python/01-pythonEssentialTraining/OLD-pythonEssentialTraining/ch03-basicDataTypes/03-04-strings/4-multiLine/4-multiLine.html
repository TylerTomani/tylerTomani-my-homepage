03-04-4 Multi line format

 You can not put \n new lines in strings without ''' triple quotes 

str = 'this should
go multiple 
lines'

Use ''' triple quotes to insert \n new lines into strings 

str = '''this should
go multiple
lines'''
print(str)

  If you do this in the terminal it will display ... and wait for input for each new line

It stopes once ''' triple quotes are entered again 

'''this should
go multiple
lines'''

Notice that when you do it this way the \n characters are displayed in the output instead.

storing in a variable then typing that variable will also display those characters 

str = '''this should
go multiple 
lines'''
str

To dislay the multiple lines, place ''' tripled quoted string into variable then print(variable)
