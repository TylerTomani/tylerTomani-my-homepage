f'' string format

  We can use the f'' string format for concatenating strings by placing variables within {} curly braces

f'my number is : {5}'

We can put expressions inside the f strings.

f'{2} times {2 * 3} equals {2 * 2 * 3}'

import math

 import the math class to see how to use math function within f strings

Pie in the math class goes to the 15th decimal place

All 15 decimal places are printed out

f'{math.pi}'

We can add :2f to limit it to decimal places

f'{math.pi:2f}'
