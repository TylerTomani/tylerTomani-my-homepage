03-04-1 slice

slice() list and strings

Slicing strings and list works very much the same 

name = 'My name is Larry'
name[0:2]
my_list = [1,2,3,4]
print(my_list[0:2])

Accessing their indexes works simliarly as well. 

name = "My name is Larry"
name[1]
my_list =  ['one','two','three']
my_list[1]

[first argument: second argument], slices from first to second argument

No first argument implies [0] or the start of my_list or name

This will take the first 2 characters list items. 

name = 'My name is Larry'
name[:2]

This will take the first 2 list items 

my_list = [1,2,3,4]
my_list[:2]

No second argument implies the end of my_list or name

This will take the characters from 11 to the end 

name = 'My name is Larry'
name[11:]

This is will take the last 3 list item
