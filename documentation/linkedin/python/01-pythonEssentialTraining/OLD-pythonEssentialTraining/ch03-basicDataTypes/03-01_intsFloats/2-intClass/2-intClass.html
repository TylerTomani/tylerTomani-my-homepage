03-01-ints and floats 2_int() Class

Python has built in classes written in lowercase.

 int() is a built-in class that converts floats into integers. This process is called casting. int() looks like a funciton but it is a built-in class.

 This converts the result to an int rather than a float. 

exp = int(3 * 4.0)
print(type(exp), exp)

 Casting a floating point number into the int class will truncate the decimal and not round it.

print(int(8.999))

Imporant note,
int will not round with division either, if there are remaining decimal places

print(int(14 / 3))
