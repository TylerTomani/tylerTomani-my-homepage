 We can modify the code above seen in the last lesson to store the results of the function into a data structure such as a dictionary in order to display the results of the two threads.

 We do this by creating a empty dictionary and within the function put the result into that dictionary. We must also supply the dictionary results into the args= within the second arguement of Thread(). 

import threading
import time

def longSquare(n,results):
    time.sleep(0.5)
    results[n] = n**2

t1 = threading.Thread(target=longSquare, args=(1,results))
t2 = threading.Thread(target=longSquare, args=(2,results))
t1.start()
t2.start()
t1.join()
t2.join()
print(results)
