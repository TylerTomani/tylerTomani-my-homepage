02-04-1_if/Else

The if statement

a = True
if a:
    print('a is true')

if a is the same as saying if true.

 If the condition of the if is true, the code on the following line will execute but only if it is indented.

 The block is the indented line of code on the next line following the :

The else statement

a = 1
b = 2
if a == b:
    print('true')
else:
    print('false')

The code after an else statement runs if the original if statement is not ture.

We can nest if statments within if statment with proper indentation.

Notice how the last if c: statement does not execute because the previous if b: statement is False which exits the original if statement 
a = True
b = False
c = True
if a:
    print('a is true')
    if b:
        print('a and b are true')
        if c:
            print('all true')
else:
    print('nothing is true')
print('but all okay')

Once a condition is met or not met, the program moves on to the code after the if or if / else statement, seen with

print("but all okay") print statement

If we change b to True, all the if statement's indented blocks of code will execute 

a = True
b = True
c = True
if a:
    print('a is true')
    if b:
        print('a and b are true')
        if c:
            print('all true')
else:
    print('nothing is true')
print('but all okay')

If we change a to False, the else statement is executed along with the proceeding print statement 'but all okay'.
