02-04-2_forLoop

For loop iterates throught what python called iterables such as;

  • list
  • set
  • tuples
  • dictionaries

 To iterate through a list with a for loop use the for item in list

The word item here can be named anything. It is assigned to the current item of the list starting from the first item, and incrementing up to the last. 

my_list = [1,2,3,4]
for item in my_list:
    print(item)
