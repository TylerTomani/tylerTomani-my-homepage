02-03-logicalOperators

There are three Logical operators

  • and
  • or
  • not

and

 Both sides must evaluate to True in order to produce a True result.

print(True and True)

I have to fix this image

print(True and False)

Problem with and

It seems that and will only return True or False if the values being compared are either True or False

 This can be seen when comparing to variables with the same value. Only the last value is returned 

dog1 = 'Skippy'
dog2 = 'Skippy'
print(dog1 and dog2)
dog1 = 'Skippy'
dog2 = 'Rooster'
print(dog1 and dog2)

or

This evalutes to True if either condition is True.

print(True or True)

The only time it evalutes to False is if both conditions are false.

print(False or False)

not

The not operator only acts on one value and simply flips the boolean.

print(not True)
print(not False)
