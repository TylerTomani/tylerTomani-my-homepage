02-02-2_Sets

 Sets are almost indentical to list except each element has to be unique.

We define a set with { } curley brackets

my_set = { }

Order does not matter in a set.

{1,2} == {2,1}

Assign values to a set just like a list, but with {} brackets instead of [] square one. 

my_set = {1,2,3,4}
print(my_set)

We can use the type() function to see that it is a set 

my_set = {1,2,3,4}
print(type(my_set))

The len() function also works on sets. 

my_set = {1,2,3,4}
print(len(my_set))

If we fill a set with mutiple same values, the len() function will only return the number of unique values. 

my_set = {1,1,2,2}
print(len(my_set))

Since sets can only contain unique values, the value return from len() is 2 and not 4.

And if we print() my set, it will only display the unique values 

my_set = {1,1,2,2}
print(my_set)
