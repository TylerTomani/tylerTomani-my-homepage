Order of Elements in List vs Sets

The order of the elements in a list is very important

The order of the elements in sets does not matter.

If we compare two list with the same values in a different order,

print([1,2] == [2,1])

It will return False

If we compare to sets with values in a different order

print({1,2} == {2,1})

It will be True
