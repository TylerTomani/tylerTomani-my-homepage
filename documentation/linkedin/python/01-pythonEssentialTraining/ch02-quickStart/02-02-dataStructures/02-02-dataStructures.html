02-02-dataStructures

  Python uses different types of data structures to store multple values into one variable.

The structures include:
  • List [ ]
  • Sets { }
  • Tuples ( )
  • Dictionaries {
     'key': 'value'
    }

 There are different methods used to work with these data structures

 The methods that are discuseed in this lessons include:

  • len() - returns the length of a structure
  • type() - shows the type of variable or structure
  • append()- add items to a structure
