 We assign variables available only within the scope of the class by using self. 

class Dog:
    def __inti__(self):
        self.name = 'Rover'

 When defining functions within the class we pass in self as an argument.

 This allows the function to use the variables from within the class. 

class Dog:
    def __init__(self):
        self.name = 'Rover'
        self.legs = "4"
    def speak(self):
        print(self.name + " has " + self.legs + " legs")
