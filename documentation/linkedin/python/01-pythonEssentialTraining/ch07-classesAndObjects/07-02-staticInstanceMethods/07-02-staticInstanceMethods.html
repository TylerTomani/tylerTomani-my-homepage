07-02-staticInstanceMethods

This class appends words to a sets. It puts the txt through two class methods. addTxt and cleanTxt 

class WordSet:
    def __init__(self):
        self.words = set()
    def addTxt(self,txt):
        txt = self.cleanTxt(txt)
        for word in txt.split():
            self.words.add(word)
    def cleanTxt(self,txt):
        txt = txt.replace('!','').replace('?','')
        return txt.lower()
wordSet = WordSet()
wordSet.addTxt('? These are some!! words?')
wordSet.addTxt('!!!! These are more words!?!')
print(wordSet.words)

Don't forget the split() function

The self in cleanTxt(self,txt) is Not being used. 

    def cleanTxt(self,txt):
        txt = txt.replace('!','').replace('?','')
        return txt.lower()

But, if we try to remove it and run the code, it will throw an error. 

    def cleanTxt(txt):
        txt = txt.replace('!','').replace('?','')
        return txt.lower()

We can omit self from cleanTxt(txt) by adding a @staticmethod decorator above the cleanTxt() method 

class WordSet:
    def __init__(self):
        self.words = set()
    def addTxt(self,txt):
        txt = self.cleanTxt(txt)
        for word in txt.split():
            self.words.add(word)
    @staticmethod
    def cleanTxt(txt):
        txt = txt.replace('!','').replace('?','')
        return txt.lower()
wordSet = WordSet()
wordSet.addTxt('? These are some!! words?')
wordSet.addTxt('!!!! These are more words!?!')
print(wordSet.words)

 cleanTxt() is a static method because it doesn't use self, while addTxt() is an instance method because it requires the instance of the class

To fix this we can use the @staticmethod decorator above the cleanTxt() method. Add also place the punctuations we wish to replace in a static attribute 

class WordSet:
    puncs = ['!','?']
    def __init__(self):
        self.words = set()
    def addTxt(self,txt):
        txt = self.cleanTxt(txt)
        for word in txt.split():
            self.words.add(word)
    @staticmethod
    def cleanTxt(txt):
        for punc in WordSet.puncs:
            txt = txt.replace(punc,'')
        return txt.lower()
wordSet = WordSet()
wordSet.addTxt('? These are some!! words?')
wordSet.addTxt('!!!! These are more words!?!')
print(wordSet.words)
