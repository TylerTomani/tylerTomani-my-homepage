07-03- Inheritance

We will use the Dog class as the parent class. 

class Dog:
    _legs = 4
    def __init__(self,name):
        self.name = name
    def speak(self):
        print(f'{self.name} says Bark')
    def getLegs(self):
        print(f'{self.name} has {Dog._legs} legs')
myDog = Dog('Rover')
myDog.speak()
myDog.getLegs()

 Place pass child class and it will have all attributes and methods from the parent.

Put Dog within the parenthesis of the child class 

class Chihuahua(Dog) :
    pass

The class Chihuahua has access to all the attributes and methods from dog 

dog = Chihuahua('lola')
dog.speak()

 We can change/extend a method from the parent class by creating the same method within the child class. The child class method take precedence over the parent's method of the same name.

  We can also add methods to the child class. You can see the Chihuahua class speak() method now says ruff ruff, and has the new method wagTail() 

class Chihuahua(Dog):
    def speak(self):
        print(f'{self.name} say bark bark')
    def wagTail(self):
        print('wagging tail')
dog = Chihuahua('Roxy')
dog.speak()
dog.wagTail()
