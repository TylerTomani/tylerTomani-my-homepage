If the number is divisible by 3 print 'Fizz', divisble by 5 print 'Buzz' and if the number is divisible by 15 print 'FizzBuzz', other wise just print the number 

for n in range(1,50):
    if n % 3 == 0:
        print('fizz')
    else:
        if n % 5 == 0:
            print('buzz')
        else:
            if n % 15 == 0:
                print('fizzbuzz')
            else:
                print(n)

This code works but it messy. Python has the elif statement which will clean it up. The else statement at the end is optional. 

for n in range(1,50):
    if n % 15 == 0:
        print('FizzBuzz')
    elif n % 5 == 0:
        print('Buzz')
    elif n % 3 == 0:
        print('Fizz')
    else:
        print(n)

Single if statement line

You can write an if statement on one line, In Python it does NOT seem like you can use an if without an else in a print function 

n = 3
print(' \'fizz\' ' if n % 3 == 0 else n)

Ternary Operator

we can apply this one line if statement as a ternary operator to assign a value to a variable 

n = 7
oddNum = 'Even Number' if n % 2 == 0 else 'Odd Number'
