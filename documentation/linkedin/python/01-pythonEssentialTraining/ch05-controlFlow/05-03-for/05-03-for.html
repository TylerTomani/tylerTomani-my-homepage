continue

 Use the continue method to skip a block of code. Great way of using exit clause first.

 This example skips any list containing more than one animal and prints only the first animal in the list. 

animalDict = {
'a': ['aardvark','antelope'],
'b': ['bear'],
'c': ['cat'],
'd': ['dog']
}
for letter,animals in animalDict.items():
    if len(animals) > 1:
        continue
    print(f'only one animal : {animals}')

break

We can break to exit out of a loop if we find what we are looking for. So in this example, the loop will print any list with more that one item then break 

animalDict = {
    'a': ['aardvark','antelop'],
    'b': ['bear'],
    'c': ['cat'],
    'd': ['dog']
}
for letter,animals in animalDict.items():
    if len(animals) > 1:
        print(f'found {len(animals)} animals: {animals}')
        break
