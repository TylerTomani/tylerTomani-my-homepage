04-04-01- List Comprhension & Filters

Perform an action on each item in a list by doing the following. This multiples each element by 2 

myList = [1,2,3,4,5]
[2*item for item in myList]

 The first part 2*item is the action performed on each element from the list and gets returned.

 typing item alone returns each item from the list. 

[item for item in myList]

We can return only items that meet a condition by adding an if at the end 

myList = list(range(1,22))
[item for item in myList if item % 3 == 0]

Cool Example, this will return only numbers ending with 0, 1, or 2 

myList = list(range(1.51))
filtered = [item for item in myList if item % 10 < 3]
filtered
