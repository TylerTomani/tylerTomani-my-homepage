04-05-01-dictComprehend

We can extract the key value pairs into a list of tuples useing the .items() method on a dicitonary. 

animalsList = [('a', 'aardvark'),('b','bear'),('c','cat'),('d','dog')]
animalsDict = {key: value for key,value in animalsList}
animalsDict.items()

.items returns a dict_items object

This returns a dict_items object containing a list of tuples consisting of the key values pairs.

We can take the dict_items object and covert that into a list with a the list() object. This will return the dictionary object back into it's original list. 

list(animalsDict.items())

We can return a list of dictionary items form the list of tuples, adding attributes in the process 

[{'letter': key , 'name': value} for key,value in animalsDict.items()]
