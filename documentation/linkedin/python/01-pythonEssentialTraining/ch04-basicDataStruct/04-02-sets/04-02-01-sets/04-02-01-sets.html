Or, us the set() constructor method that turns any iterable object into a set.

  • string
  • lists
  • sets
  • dicst
  • tuples
  • or custom class

Turn a list ['a','b','c'] into a set {'a','b','c'} 

mySet = set(['a','b','c'])
mySet

And we can change an immutable tuple ('a','b','c') into a set {'a','b','c'} 

mySet = set(('a','b','c'))
mySet
