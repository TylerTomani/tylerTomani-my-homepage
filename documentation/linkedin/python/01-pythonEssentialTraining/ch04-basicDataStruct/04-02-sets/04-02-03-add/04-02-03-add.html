04-02-03- add()

Use the add() method to append items to a [] set 

mySet = {1,2,3}
mySet.add(4)
mySet

 .add() on sets {} is like .append() on list []. The difference:

  • .append() inserts the items at the end of the list [] (in memory)
  • .add() places the item randomly in a set {}

because items are stored randomly within sets {} (in Memory).
