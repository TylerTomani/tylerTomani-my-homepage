04-02-tuples

Tuples can not be changed. 

myTuple = ('apple','banana','orange')
myTuple[1] = 'pineapple'

But they can be indexed 

myTuple = ('apple','banana','orange')
myTuple[1]

Tuple's size is set to a limit. This is why python uses them by defualt

returning values, seperated by commas, from a function; returns them as a tuple by default. 

def returnSomeValues():
return 1,2,3
returnSomeValues()

you can include the () parenthesis if you want to, but unless another data structure is typed in return a tuple will return by default 

def returnSet():
    return [1,2,3]

returnSet()

def returnTuple():
    return (1,2,3)

returnTuple()

You can also declare tuples as variables withouth the () parenthesis. 

myTuple = 1,2,3
myTuple
