04-03-03 get() when key not in

Indexing a key that doesn't exist causes a KeyError. 

animals = {
    'a': 'aardvark',
    'b': 'bear',
    'c': 'cat',
    'd': 'dog'
}
animals['e']

.get() will check if the key (first parameter) is in the dict. The second parameter returns if it is not there.

animals.get('e','e is not there')

If the key is there get() will return the value.

animals.get('a','a is not there')

If we print out the return of get() if the key is not present in the dict. None is returned

print(animals.get('e'))
