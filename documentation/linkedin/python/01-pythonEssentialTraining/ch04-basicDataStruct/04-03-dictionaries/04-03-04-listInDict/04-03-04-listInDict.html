04-03-04 list in dict

It is common to see list [] as values in dictionaries 

animals = {
    'a': ['aardvark','antelop'],
    'b': ['bear'],
}

We can add to a list as a value in a dict with append()

animals['b'].append('bison')

We can assign a new key with a new list

animals['c'] = ['cat'];

If there was a key already called c with a list. Doing the above would overwrite the old value 

animals = {
    'a': ['aardvark','antelope'],
    'b': ['bear','bison'],
    'c': ['caribou']
}
animals['c'] = 'cat'
animals

Use an if statement to check if key exists to avoid overwriting any values. The following will make a key 'c' with a empty list [] as a value if 'c' does not exist.

Otherwise it will just append 'cat' to the key 'c' if it exist 

if 'c' not in animals:
    animals['c'] = []

animals['c'].append('cat')
animals
