08-02-03 Catch exceptions by type

Catching Exception with except catches any Error. 

def causeError():
    try:
        1/0
    except Exception:
        print('error')

causeError()

But we can specify what type of Exception to catch, for example ZeroDivisionError 

def causeError():
    try:
        1/0
    except ZeroDivisionError:
        print('a zero division error ')
    except Exception:
        print('error')

causeError()

We can chain different exceptions together, but the order matters. If we place the basic Exception first, it won't catch the TypeError which is it is 

def causeError():
    try:
        1 + 'a'
    except Exception:
        print('error')
    except ZeroDivisionError:
        print('zero division error')
    except TypeError:
        print('type error')
    

causeError()

Place the most important Errors first, 

def causeError():
    try: 
        return 1 + 'a'
    except TypeError:
        print('type error')
    except ZeroDivisionError:
        print('zero division error')
    except Exception:
        print('error')


causeError()
