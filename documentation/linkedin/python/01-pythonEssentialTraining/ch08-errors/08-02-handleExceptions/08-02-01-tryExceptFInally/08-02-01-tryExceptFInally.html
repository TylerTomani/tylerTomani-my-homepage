08-02-01 Try/Except Finally

We are catching the Exception class with except. e is the instance 

def causeError():
    try:
        1/0
    except Exception as e:
        return e


causeError()

We don't need to catch the class instance e. We can do anything we want within the block of except 

def causeError():
    try: 
        1/0
    except Exception:
        print('ooops')


causeError()

finnaly allows us to execute code no matter what 

def causeError():
    try: 
        1/0
    except Exception:
        print('something happened')
    finnaly:
        print('always works')


causeError()

You do not need to use except with finnaly, but the error will be printed out instead of caught. 

def causeError():
    try:
        1/0
    finally:
        print('always works')


causeError()
