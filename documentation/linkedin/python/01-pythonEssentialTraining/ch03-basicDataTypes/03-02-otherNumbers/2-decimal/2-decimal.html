Import the Decimal class with the the decimal module by putting this line of code at the top of your script

from decimal import Decimal ,

This will import both the Decimal and getcontext class from decimal

Now print out the contents of getcontext() 

from decial import Decimal , getcontext
print(getcontext())

 It will print out all the attributes of getcontext which are the global settings that get applied to the usuage of the decimal class.

We can change any setting we want using . dot notation 

from decimal import Decimal , getcontext
print(getcontext())
getcontext().prec=4
print(getcontext())

Notice how the precision attribute prec has changed from 28 to 4

We can do math by instantiating numbers into the decimal class 

from decimal import Decimal , getcontext
getcontext().prec=4
print(Decimal(1) / Decimal(3))

As you can see the precision of the decimal is limited to 4 digits.
