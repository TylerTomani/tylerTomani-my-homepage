03-02-1-int() class - Change 'string' to int

Passing an integer as a string, int('10'), converts it into an integer.

print(type(int('100')), int('100'))

This will not work with a float

print(int('10.0'))

Use int('10',2) to change base 10 to decimal

The first arg can be any based number but Must be written as a string.

int('0002')

 The second argument is the base that the number is converted to.

print(int('0010',2))

'0010' in binary (base 2) is 2

print(int('0010',10))

'0010' in base 10 is 10.

base 10 to binary

print(int('1000',2))
print(int('0100',2))
print(int('0010',2))
print(int('0001',2))

binary base2 to base 10 

print(int('1000',10))
print(int('0100',10))
print(int('0010',10))
print(int('0001',10))
