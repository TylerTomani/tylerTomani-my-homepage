03-01-1- ints

int / int = float

Division with integers produces float type variables 

division = 20 / 4
print(type(division),division)

*you can also prints seperated variables with , between

int + float = float

Adding floats to ints creates float type results. 

addToFloat = 1 + 2.8
print(type(addToFloat), addToFloat)

int * float = float

intTimesFloat = 3 * 7.0
print(type(intTimesFloat), intTimesFloat)

exponents

int ** int = int
int ** float = float

intToInt = 4 ** 6
intToFloat = 4 ** 6.0
print(type(intToInt), intToInt)
print(type(intToFloat), intToFloat)
