03-01-4- floats

When workng with floats, results will return many decimal places
prints(1.2 - 1.0)
prints(14 / 3)

The decimal places reach the maximum computer memory allocated to numbers and round at the end.

You can use the round function to fix this

print(round(1.2 - 1.0))

Add a second argument to round() to indicate what decimal to round to.

print(round(1.2 - 1.0 , 2))
