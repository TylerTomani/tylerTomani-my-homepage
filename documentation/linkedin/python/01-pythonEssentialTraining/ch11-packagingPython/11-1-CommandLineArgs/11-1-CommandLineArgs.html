11-1- Command-Line Arguments

We can pass in arguments into a script throught the command line by using the ArgumentParser class from the argparse module 

from argparse import ArgumentParser

The ArgumentParser class creates an object which allows you to track each of the arguements typed in the command line. First create an instance of the class then use the add_argumentmethod to create an argument 

from argparse import ArgumentParser

parser = ArgumentParser()

parser.add_argument('--output','-o')

The above values '--output','o' allow the user to use either argument, the goal here for this arugment is to allow the user to input a name for the output file in the command line. for example newFile.txt.

To get newFile.txt we must use the parse_args() method and assign it to a variable. If we print the variable we will see that newFile.txt is now available in the program. 

from argparse import ArgumentParser

parser = ArgumentParser()

parser.add_arguments('--output','-o')

args = parser.parse_args()

print(args)

The above code will output all arguments available in the program, but we can specify the specific argument with dot notation on args 

print(args.output)

To make the --output required add the required argument and set it to True. We can also supply help text with the help argument to tell the user that the argument is required. By default if you supply the script with the arugment -h it will display the message 

from argparse import ArgumentParser

parser = ArgumentParser()
parser.add_arguments('--output','-o',required=True, help='ouput file name required')
args = parser.parse_args()
print(args)

To close the lesson add another argument for input then use with to write to the newFile.txt output file. 

from argparse import ArgumentParser

parser = ArgumentParser()
parser.add_arguments('--output','-o',required=True,help='outfile name required')
parser.add_argument('--input','-i',required=True,help='add text to file')

args = parser.parse_args()

with open(args.output, 'w') as f:
    f.write(args.input)
print(f'The file: "{args.output}" has the following text \n"{args.input}')
