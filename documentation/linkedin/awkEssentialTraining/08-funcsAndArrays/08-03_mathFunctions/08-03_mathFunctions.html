08-03-Math Functions

  • -int(x) - returns the integer part of its argument
    • int(3.9) returns 3
  • -rand() returns random floating point value b/w 0 and 1
    • int(rand() * 6) + 1 - returns number b/w 1 and 6
  • - srand([x]) seeds rand() with the value of x.
    • if x ommited it returns the current date and time
  • - sqrt() returns the square root of the argument
  • - sin(x) returns sine of the arugment (value in radians)
  • - cos(x) returns cosine of the arugment (value in radians)
  • - atan2(y,x) returns the arctangent of y / x

If you need the value of pi, use atan2(0,-1)

  • - log(x) returns the natural log of x
  • - exp(x) returns the expodential of x ( e ^ x)
