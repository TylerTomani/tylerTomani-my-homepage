length()

legnth("antidisestablishmentarianism", "blish"

  • Return 12, the index of b

index()

index("antidisestablishmentarianism", "blish"

  • Return 12, the index of b

match()

match("antidisestablishmentarianism", /b[a-z]*n/)

Looks for in string with a regular expression, then;

  • Returns 12
  • Sets RSTART to 12
  • Sets RLENGTH to 14
  • Returns 12, the index of b

substr()

substring("antidisestablishmentarianism", 12, 5)

The 12 is the index to start from. The 5 is how many to go

  • Returns "blish"

substring("antidisestablishmentarianism", 12, )

If the third argument 5 is omiited,

  • Returns the rest of string "blishmentarianism
