04-03-operatorsAndArrays

 awk provides three string operators

  • concatenation - doesn't have a symbol
  • ~ - checks if regular expression matches another
  • !~ - checks if regular expression does NOT match another

Arrays

  turn a variable into an array using subscript, just like other language, but remember, $0 is the whole line.
And, awk only accepts single dimension arrays

$ awk '{a[1]=$1;a[2]=$2;a[3]=$3; print a[1] , a[2], a[3]}'
dog cat mouse
dog cat mouse
