1-Callback Reading

 Functions can be stored in variables. They can also be passed into functions as arguments. This is refered to as a Callback. We can invoke these two variables like functions.


let foobar = function(callback){
 console.log("foo")
 callback();
 console.log("bar")
}
let sayHey = function(){
 console.log("hey")
}
foobar(sayhey);
// prints foo
hey
bar

 This allows for different operations to occur within a function, like this math example. We can return the result from double or negate. 

    let math = function(num1, num2, cb){
        let sum = num1 + num2;
        let result = cb(sum):
        return result
    }
    let double = function(num){
        return num * 2;
    }
    let negate = function(num){
        return num * -1;
    }
    console.log(math(2, 2, double));
    console.log(math(2, 2, negate));
// prints
8
-4

Pass in built in functions

 We can also pass in built in functions 

    let math2 = function(num1, num2, cb){
    let sum = num1 + num2;
    return cb(sum);

    }
    console.log(math2(8, 8, Math.sqrt))
//prints
4

Make the call back optional

 Javascript is NOT strict when it comes to using arguments within functions, so if you try to use a argument in a function that wasn't passed in, it just becomes undefined 

        let hiNames = function(name1, name2){
            console.log("Hey " + name1 + ", and Hey " + name2)
        }
        hiNames("Ted","Mary")
        hiNames("Bob")
        
        // prints Hey Ted, and Hey Mary
        // prints Hey Bob, and Hey undefined

 We can take advantage of undefined and use an if statement to run a function without the callback.

If the callback is undefined, run the normal function. Else, run the callback 

        let math3 = function(num1, num2, cb){
            let sum =  num1 + num2;
            if(cb === undefined){
                return sum;
            }
            return cb(sum);
        }
        console.log(math3(5,5))
        console.log(math3(5,5,Math.log))
