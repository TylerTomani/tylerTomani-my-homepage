02-01_Writing an Awk program

The basic syntax works as follows

  • awk
  • the program within '{ }'
  • text file to run awk on

 AWK, is best suited to work with txt.files. Each line of a text file is a record. Every record has a series of fields. The fields are seperated by the seperator By default, the seperator is blank space between each field.

 Each field in a record is numbered numerically from left to right. They can be accessed in the program using $1, $2, etc ...

 The following awk program switches the position of last name $2 with the position of the first name $1, seperated by a space. A comma by itself between the fields make the output seperator a space

This is very frustrating, the code from the video only changes the first line. The reason is, there are too many spaces in between the fields, the default seperator must be a single space

The program should change every name but it is only working with the first line, as seen in the red box

 To use awk without a text file, type in awk followed by the command,
this will open the command prompt. This command will print out the first 2 fields of the record/line. Notice how there is no space in the output

 To create a space between the fields in the output, add a , between the field placeholders $1 and $2
