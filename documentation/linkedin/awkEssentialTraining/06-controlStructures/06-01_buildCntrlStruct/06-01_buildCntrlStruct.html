This example prepends the string Short Line: to any record containing < 8 fields, and Long Line: to any other

 If writing an if statement on one line, don't loose track of the last closing bracket }

$ awk '{if (NF < 8) {print "Short Line: " , $0} else {print Long Line: , $0}}' poem.txt
Short line: The grand old Duke of York
Short line: He had ten thousand men
Long Line: He marched them up to the top of the hill
Short line: And he marched them down again
Long Line: And when they were up they were up
Long Line: And when they were down they were down
Short line: And when they were only half-way up
Short line: They were neither up nor down

 It is best to write if statements in a file and save that file with file.awk. 

linelength.awk
{
    if (NF < 8){
        print "Short Line:" , $0
    } else {
        print "Long Line:" , $0
    }
}
$ awk -f linelength.awk dukeofyork.txt

This will produce the same result as above
