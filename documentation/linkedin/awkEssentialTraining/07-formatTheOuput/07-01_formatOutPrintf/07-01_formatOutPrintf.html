 The following code shows how to use printf() to reformat the above csv "Comma Seperated "input file to the above tsv "Tab Seperated" awk output file.

 This example only changes the format without specifying the space between fields, (see next lesson for that)

 *Important note, printf() does NOT automatically add a new line \n to the end of the output. AND, it does NOT use OFS or ORS

Notice how $3 field three has only two digits. \d cuts the decimals and converts it into an integer. 

 awk -F , '{printf("%s\t%s\t%d\n",$1,$2,$3)}'
