If you use the -F flag on ls, it will use special characters to display what type of file each item in the directory is.

  • / for directory
  • * for and executable file

For practice purposes make an output files of all the items within the direcotry

$ ls -lF > ls.out

If we use the same command from above to add up all the memory of .txt files, it will miss any executable ones because of the asterisk *.

$ awk '/\.txt/{total += $5}END{print total}' ls.out
2442
